<Card title="Understanding AI Agents" icon="robot">
  An agent is a specialized software program that can perform specific tasks
  autonomously. Think of it as your digital assistant like Siri, Alexa, or
  Google Assistant that can interact with various applications, APIs, and
  services to get things done on your behalf.
</Card>

## How Agents Work in Bhindi

<Info>

## How to Use Agents

<Card title="Natural Language Interface" icon="microphone">
  Simply tell Bhindi what you want in plain English - no complex commands or
  syntax required!
</Card>

### Examples of Natural Language Commands

```text
"Send a message to the #general channel in Slack"
```

```text
"Create a new page in my Notion workspace"
```

```text
"Schedule a meeting for tomorrow at 2 PM"
```

```text
"Summarize the latest emails in my inbox"
```

<Note>
  The more specific you are with your requests, the better agents can understand
  and execute your tasks.
</Note>

## Getting Started with Agents

<Steps>
  <Step title="Explore Available Agents" icon="magnifying-glass">
    Browse the [agent directory](https://directory.bhindi.io/) to see what's available and discover agents that match your workflow needs.
    
    <Card title="Agent Directory" icon="grip" href="https://directory.bhindi.io/">
      Discover all 80+ available agents and their capabilities
    </Card>
  </Step>

<Step title="Try Built-in Agents" icon="play">
  Start with pre-built agents for common tasks like email management, calendar
  scheduling, or document creation.
  <Tip>
    Begin with agents for tools you already use daily - this will give you
    immediate value and help you understand how agents work.
  </Tip>
</Step>

<Step title="Connect Your Tools" icon="link">
  Set up authentication for your services to allow agents to access and interact
  with your accounts securely.
  <Note>
    All connections use industry-standard OAuth or API key authentication for
    maximum security.
  </Note>
</Step>

<Step title="Build Custom Agents" icon="hammer">
  Once comfortable with built-in agents, create custom agents for your specific
  needs and internal tools.
  <Card title="Custom Agent Guide" icon="book" href="/custom-agent/quickstart">
    Learn how to build your first custom agent
  </Card>
</Step>

  <Step title="Automate Workflows" icon="bolt">
    Chain agents together for complex automation and create powerful multi-step workflows.
    
    <Warning>
      Start simple with single-agent tasks, then gradually build more complex workflows as you become comfortable with the system.
    </Warning>
  </Step>
</Steps>
